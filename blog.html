Important FAQ

What is the Document Object Model (DOM)?

The DOM is a programming interface that represents a document's structure as a tree of objects, allowing for dynamic manipulation of its content and structure.

How do you select an element from the DOM?

Use methods like:
document.getElementById('myId')
document.getElementsByClassName('myClass')
document.getElementsByTagName('div')
document.querySelector('.myClass')

What is event delegation in the context of the DOM, and why is it useful?

Event delegation attaches a single event listener to a parent element to manage events from child elements. It's useful for performance, handling dynamic content, and simplifying code.

How do you manipulate an element's attributes and styles using the DOM?

Attributes:
Get: element.getAttribute('attrName')
Set: element.setAttribute('attrName', 'value')
Remove: element.removeAttribute('attrName')

Styles:
Set inline: element.style.propertyName = 'value'
Add Class: element.classList.add('className')
Remove Class: element.classList.remove('className')
Toggle Class: element.classList.toggle('className')
